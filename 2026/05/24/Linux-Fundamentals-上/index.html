What is the path to htb-student’s home directory?

直接pwd查看当前所在目录就行

What is the path to the htb-student’s mail?

两种方法看:

  1. 通常在/var/mail/ 或 /var/spool/mail/

  2. 或者看环境变量,env | grep MAIL

Which shell is specified for the htb-student user?

直接uname -r
